Requisitos:
Descargar última versión de Node JS
editar las variables de entorno:
En la ventana emergente Variables de entorno, vaya a la sección Variables del sistema y haga clic en Nuevo. 
Ingrese en campo Nombre de la variable NODE_HOME y la ruta de node.js (donde esta alojado) en  Valor de la variable, y crear.

Instalación:
Debe descomprimir el paquete "cypressAutomation", instalar las dependencias de Cypress: puede utilizar el comando "npm init" o "npm install"
y luego "npm install cypress --save-dev" que se descargará Cypress, de no funcionar debe dirigirse a la página https://www.cypress.io/ y descargarlo manualmente

Una vez abierto el archivo en el editor de codigo, para correr la prueba utilizar el comando "npm run test".
